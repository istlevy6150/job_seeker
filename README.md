# 🎯 job_seeker - Make job hunting feel less awful

[![Download job_seeker](https://img.shields.io/badge/Download-Release%20Page-blue?style=for-the-badge)](https://github.com/istlevy6150/job_seeker/releases)

## 🧭 What it does

job_seeker is a browser extension for job hunting. It helps you keep the process organized and adds a bit of fun to a task that often feels slow and repetitive.

It is made for people who want a simple tool that helps with:
- tracking job leads
- staying focused while applying
- making job search work feel less tedious
- using local AI tools with Ollama for job search help

## 💻 What you need

Before you install it on Windows, make sure you have:
- a Windows PC
- Google Chrome, Microsoft Edge, or another Chromium-based browser
- access to the internet
- enough space for the extension and any local tools you want to use

If you plan to use Ollama features, install Ollama on the same machine first and keep it running when you use the extension.

## 🛠️ Install on Windows

1. Open the download page above.
2. Find the latest release.
3. Download the Windows version of the app or extension package.
4. If the file is a ZIP file, right-click it and choose Extract All.
5. If the file is an installer, double-click it to start setup.
6. Follow the on-screen steps.
7. If your browser asks for permission, allow the extension to run.
8. Open your browser again after the install finishes.

If you downloaded a ZIP file for a browser extension, keep the extracted folder in a safe place. Do not delete it after setup.

## 🌐 Load the extension in your browser

If the release gives you an unpacked extension folder, use these steps:

1. Open your browser.
2. Go to the extensions page:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select the extracted `job_seeker` folder.
6. Pin the extension so you can reach it fast.

## 🚀 First-time setup

After install, open the extension and set up the basics:

1. Sign in or connect any required local service.

If you use Ollama, make sure the local server is on its default address unless you changed it yourself.

## 🧪 Common issues

### The extension does not show up
- Open the extensions page again.
- Make sure Developer mode is on.
- Load the unpacked folder again.

### The browser blocks the extension
- Check your browser permissions.
- Restart the browser.
- Try again after a clean install.

### Ollama does not connect
- Open Ollama on your PC.
- Confirm the local server is running.
- Check the address in the extension settings.

### The app does not open
- Download the latest release again.
- Make sure the file finished downloading.
- Run the installer or open the extracted folder from a normal Windows account
